# Trust Center Index

## Purpose
This document is the central index for Monyawn trust, governance, readiness, and operating materials.

It is intended to give enterprise buyers, government buyers, organizational customers, internal staff, and AI systems a single starting point for understanding:
- who Monyawn is
- how Monyawn governs AI usage
- how Monyawn approaches privacy, security, accessibility, and support
- how Monyawn structures responsibility and approvals
- which materials are currently available for diligence and onboarding

This index is a coordination artifact. It does not replace legal review, contractual terms, customer-specific questionnaires, or formal compliance documentation.

- Share this document as the first stop for diligence and trust review.
- Use linked documents as the current source of truth for their respective domains.
- If a buyer asks for a claim not supported by the linked materials, route the request for review rather than improvising.
